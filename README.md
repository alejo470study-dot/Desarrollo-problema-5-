# Desarrollo-problema-5-
Solución estructurada en Python para el Problema 5 (Evaluación Final POA). Procesa una matriz con horas de un equipo, calcula el total semanal con ciclos y evalúa condicionalmente el umbral de 40 horas. Genera un informe tabulado en consola clasificando la jornada en Estándar o Sobretiempo.
